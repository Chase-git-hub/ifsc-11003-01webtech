Project Revision and Reflection

Developing this Mars Rover website project helped me strength my understanding of the front end web development and the importance of accessibility and structure in my design. My goal was to create a clean and informative and visually appealing site that highlights NASAs Mars Rover Program and its incredible technological achievements.

During revision I focused on fixing the usability and presentation. I reorganized my HTML structure to use elements like <header>, <main>, and <footer> to make the site more accessible for screen readers. I also optimized my images by resizing and balancing their width and height to get a better visual layout across different screen sizes.

Adding a dark mode toggle was one of my favorite improvements because it allows users to switch themes for better comfort and readability. I also replaced the old email link with a real contact form so visitors can submit messages directly through the website. This change improved functionality and gave the site a more professional user focused design.

Overall this project taught me how small details like consistent navigation or color contrast and descriptive alt text and clean CSS can make a big difference in the experience. I learned how to apply technical and creative skills to present accurate and educational information in a way thats engaging and accessible to all audiences.
